# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please do not open a public issue.

Instead, report it privately by contacting the repository owner/maintainer.

Include:
- a description of the issue and impact
- steps to reproduce
- affected versions/commits (if known)
- any suggested mitigation

## Supported Versions

This project is pre-release. Only the latest commit on the default branch is supported.
